# 🏢 AI Office Dashboard

> Lucy 團隊的可視化辦公室 — 讓 AI agents 變成可愛的像素小員工

## 📖 專案概述

將 OpenClaw 的 main + sub-agent 架構視覺化，用像素風格的動畫辦公室呈現每個 agent 的工作狀態、任務統計和「員工心聲」。

## 🎯 核心功能

### 1. 像素辦公室場景
- 每個 agent 是一個可愛的像素角色
- 工作時有打字/思考動畫
- 閒置時有喝咖啡/發呆動畫

### 2. 儀表板 (三個 Tab)
- **成本節省**：總節省金額、等同全職員工數、任務統計
- **互動統計**：對話數、Token 用量、API 花費
- **員工心聲**：動態生成的角色吐槽

### 3. 角色卡系統
- 像素頭像
- 名稱 + 職稱
- 角色描述
- 月薪（虛擬）
- Skills 標籤

### 4. 排行榜
- 任務完成數
- 節省金額
- 工時統計
