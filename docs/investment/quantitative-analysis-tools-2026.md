## 🇹🇼 台股量化分析工具

### 主要選擇：TEJ 台灣經濟新報

**免費方案**：
- **資料來源**：TWSE/TPEx 官方資料，雙重驗證確保準確性
- **支援語言**：Python, R, C++, .NET
- **涵蓋範圍**：EOD 股價、歷史指數、財務報表、法人進出、企業行動
- **輸出格式**：CSV, XML, FTM

**特色**：
- 台股專屬資料集（TAIEX、台積電相關度指標）
- 支援自動化 API 整合
- 歷史深度足夠長期分析
- 機構級資料品質

### 替代選擇：TWSE 官方工具

**TWSE 平台工具**（免費）：
- ETF 篩選器
- 投資專家推薦
- 視覺化分析工具
- 基本量化篩選（不需編程）

**限制**：無直接 API，需透過核准資料商

## 🇺🇸 美股量化分析工具

### 🏆 推薦組合：Finviz + Python

| 工具 | 免費功能 | 付費升級 | 適用場景 |
|------|----------|----------|----------|
| **Finviz** | 強大篩選器（P/E、技術指標）<br/>熱力圖、新聞聚合 | Elite $39.50/月<br/>24年回測資料 | 主要篩選工具 |
| **Koyfin** | 500+ 指標、10年歷史<br/>自訂公式、百分位排名 | 付費解鎖更多指標 | 深度分析 |
| **Quiver Quantitative** | 國會交易、機構持倉<br/>免費回測器 | - | 另類資料 |
| **Yahoo Finance** | 基本財務、簡單篩選 | - | 資料補充 |

### Python 整合建議

**免費資料來源**：
- `yfinance`：Yahoo Finance API（有速率限制）
- `Alpha Vantage`：免費層級有限制

**回測框架**：
- `Backtrader`：專業回測引擎
- `Zipline`：Quantopian 開源版本
- `PyAlgoTrade`：事件驅動回測
- `pandas`：自建投資組合優化
